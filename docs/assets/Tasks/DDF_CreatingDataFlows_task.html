Creating Data Flows

A data flow connects two data storesdata stores such that files can be transferred between them.

You select the two data stores and then specify how the files are to be transferred.

  1. Click the Add a Data Store button to display the Data Flow page.
  2. Give your data flow a name.
    This is the name used to identify this data flow throughout the system
  3. Specify one or more actions.
    Actions are operations you can have the system carry out on the data during the transfer.
    See Actions.
  4. Specify recurrence information.
    You can configure when and how oftern the data flow runs.
  5. Select From and To data stores.
  6. Save your data flow.
